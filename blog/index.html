Introduction

This article will cover the fundamentals of TypeScript. To begin, what is TypeScript? TypeScript is a superset of JavaScript. This means that it is a language that extends JavaScript. We use it instead of JavaScript because it adds an additional layer of security via type checking and static checking.

This means that as a pre-requisite to getting the most out of this article, you should have a basic understanding of the fundamentals of JavaScript, such as variables, functions, objects, etc.
